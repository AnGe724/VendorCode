GeoPhoto Punch Tool

Version 1.2

Somroli's GeoPhoto Punch Tool can be used for tracking employees attendance. Employees can use this application to clock in/out their attendance. Each punch activity is captured with accurate geographic location (GeoLocation) with option to capture employees photo with every punch. Both GeoLocation and punch photos are available for review in various attendance reports. The application can be used by employees to submit leave requests. Supervisors can use this application to monitor real time attendance of one or multiple work locations. This application requires active account with Somroli's cloud based time and attendance software.
